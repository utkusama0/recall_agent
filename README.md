# RECALL — Spaced Repetition PWA

A free, iPhone-compatible spaced-repetition app that runs entirely on GitHub Pages. No Anki, no subscriptions. Claude generates flashcards from your study sessions; FSRS scheduling runs in-browser; cards and state live in your GitHub repo.

---

## What it does

- **Generate cards with one command** — run `/recall-cards` in Claude after a study session; cards are written to `cards/*.json` and pushed automatically
- **Review on any device** — open the GitHub Pages URL; works offline via service worker
- **FSRS scheduling** — ts-fsrs runs entirely in the browser, zero AI tokens for scheduling
- **In-app editing** — edit any card directly in the PWA; changes sync back to GitHub
- **AI tutor** — tap "Ask" on any card to chat with Groq Llama 3.3 70B (free) or Claude via a proxy
- **Language card TTS** — tap 🔊 to hear pronunciation; IPA shown inline
- **Deck hierarchy** — browse decks grouped by domain (CS / Finance / Language)

---

## Quick start (your own fork)

1. **Fork** this repo on GitHub
2. Edit `config.json` → set `github.owner` to your username (use `config.example.json` as reference)
3. Enable **GitHub Pages** → Settings → Pages → Source: `main` branch, `/ (root)` folder
4. Get a **GitHub PAT** with `Contents: read & write` scope on this repo only
5. Open `https://YOUR_USERNAME.github.io/recall_agent/web/`
6. Go to **⚙ Settings** → enter your GitHub owner, repo, PAT → **Save**
7. Optionally add a **Groq API key** (free at [groq.com](https://groq.com)) for the AI tutor

That's it. Cards generated by the skill appear immediately after push.

## Daily workflow

```
1. Study a topic with Claude
2. Run: /recall-cards deck:CS::OSTEP::ch28-locks src:OSTEP-ch28
3. Open the PWA → Review tab → work through due cards
4. Tap "Ask" on any card you don't understand → AI tutor explains
```

## Card types

| Domain | Special fields | Notes |
|--------|---------------|-------|
| **CS** | `code`, `math`, `application` | Mandatory: complexity, invariant, trace, failure-mode, code-pattern cards |
| **Finance** | `application`, `math` | Mandatory: risk card per concept. No investment advice. |
| **Language** | `ipa`, `example` | `lang::xx` tag routes TTS; reversed cards for vocab |

## Forking for a friend

1. Friend forks the repo
2. Edits `config.json` with their own `github.owner` and `github.repo`
3. Deletes `cards/` content and `state/scheduler.json` content (start fresh)
4. Enables GitHub Pages
5. Follows Quick start steps 4–7

Each fork is independent — separate cards, separate FSRS state, separate GitHub repo.

## Security

- **No hardcoded keys** — all secrets (GitHub PAT, Groq key) are entered in Settings and stored in `localStorage` on your device only
- **Claude models** require a proxy URL (Cloudflare Worker with your Anthropic key server-side); direct browser access is blocked
- **DOMPurify** sanitizes all card content before rendering; CSP header restricts external connections
- **GitHub PAT scope** — grant `Contents: read & write` on this repo only, not all repos
